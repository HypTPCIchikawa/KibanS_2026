\noindent
\textbf{（概要）}\\
%begin 研究目的及び研究計画の概要空行付き ＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
	本研究の目的は、極限状態の超高密度天体である中性子星の内部構造を決定づける最大の難問「ハイペロンパズル」の解決に向け、ハドロン加速器実験を用いた独創的な新手法により、ハイペロン（$\Lambda$粒子）と核子の間に働く短距離相互作用を実験的に完全解明することである。
	
	理論的には、真空中における$\Lambda N$間の引力獲得メカニズムとして、$\Lambda$粒子が中間状態でストレンジクォークを含む$\Sigma$粒子へと遷移しながら2つの$\pi$中間子を交換する「$2\pi$交換プロセス」が有力視されている。しかし、中性子星深部のような高密度媒質中においては、核子のパウリ排他律によってこの引力プロセスが強く抑制（パウリブロッキング）され、結果として引力が消滅し、クォークの重なりに起因する短距離からの強い斥力が支配的になるという仮説が提唱されている。
	
	本研究では、この斥力化シナリオの正否に最終決着をつけるため、J-PARC等のビームラインを拠点として相補的な研究を展開する。(1) $\Sigma N$カスプ分光（J-PARC E90実験）による$\Lambda N$-$\Sigma N$間のチャンネル結合力の高精度決定、(2) $\alpha\Lambda$フェムトスコピー（J-PARC E88実験およびRHIC-STAR実験）を用いた、原子核中で特異的に中心密度が高い$\alpha$粒子内部の極小ソースによる短距離斥力芯の直接抽出、および(3) カイラル有効場の理論（Chiral EFT）に基づく先進的なバリオン多体系モデルへの統合解析である。もし検証により、斥力化メカニズムのみではハイペロンパズルを説明し得ないという結論に至った場合、中性子星深部におけるクォーク物質への相転移の発生が極めて濃厚となる。本研究はハドロン間相互作用の根源的理解を通じて、極限物質観の劇的な変革に迫る挑戦的な試みである。

\subsection{本研究の学術的背景や着想に至った経緯、研究課題の核心をなす学術的「問い」}
\subsubsection{学術的背景：中性子星とハイペロンパズルの陥穽}
宇宙に浮かぶ巨大な原子核とも呼ぶべき天体「中性子星」の中心部には、標準原子核密度（$\rho_0 \sim 0.17\,\text{fm}^{-3}$）の数倍に達する極限環境が実現している。内部に向かって密度が高まり核子のフェルミエネルギーが増大すると、ストレンジクォークを含むハイペロン（特に$\Lambda$粒子）を生成した方が系の全エネルギーが低くなるため、星の中心部でのハイペロン出現は熱力学的な必然となる。これまでの$\Lambda$ハイパー核精密分光研究から、標準密度において$\Lambda$粒子が感じるポテンシャルの深さは約 $-30\,\text{MeV}$ であることが確定しており、$\Lambda$粒子と核子間には明確な引力が働くことが実証されている。したがって、$\Lambda$粒子は星の中心部に極めて出現しやすい状態にある。

しかし、標準理論モデルに基づく状態方程式（EoS）の予測によれば、ハイペロンの出現はフェルミ圧を減少させ、状態方程式を劇的に軟化させる。その結果、天体観測によって確固たる証拠が得られている太陽質量の2倍（$2M_\odot$）もの重い中性子星の重力を支えることができず、ブラックホールへと崩壊してしまうという深刻な理論的矛盾が生じる。これが広く知られる「ハイペロンパズル」である。ミクロな相互作用に基づき、この天体スケールの矛盾を根本的に解決しなければならない。

\subsubsection{パズル解決に向けた2つの有力な仮説と着想に至った経緯}
現在、このハイペロンパズルを解決し得るシナリオとして、国際的に以下の2つの有力な仮説が議論されている。
\begin{enumerate}
    \item \textbf{相互作用の斥力化シナリオ}：標準密度においては引力的である$\Lambda N$相互作用が、中性子星深部のような高密度環境下においては強い「斥力」へと転化するため、出現自体が抑制されるか硬いEoSが維持されるという仮説。これを立証するためには、高密度媒質中で相互作用が斥力化するメカニズムを定量的かつミクロに解明する必要がある。
    \item \textbf{クォーク物質への相転移シナリオ}：ハイペロン出現直前にハドロンからの非閉じ込め相転移が起き、解放されたクォーク物質が形成されることで圧力を維持するという仮説。
\end{enumerate}

本研究の核心的な問いは**「核物理は中性子星におけるハイペロンパズルを解くことができるのか？」**にある。極限環境でのクォーク相転移シナリオは魅力的であるが、中性子星深部の「低温・超高密度環境」を地上実験室で直接再現し相転移を証明することは困難である。そこで申請者は**「高密度環境で$\Lambda$粒子が強い斥力を感じ得るのか？」**という検証可能な問いに着想を得た。加速器実験の精密データに基づき斥力化シナリオの正否を徹底検証し、もし斥力化のみで重い星を支えられないと判明した場合は、状況証拠的にクォーク相転移シナリオの正当性を極めて高く裏付ける。斥力芯の極限探求こそが、相転移立証に向けた最も確実かつ鋭利なアプローチである。

\subsection{本研究の目的、研究内容、独自性と創造性、どこまで明らかにするか}
\subsubsection{本研究の目的：相互作用の短距離成分の解明}
本研究の目的は、真空中および高密度媒質中における$\Lambda N$相互作用の短距離成分を実験的に完全解明し、ハイペロンパズルに決着をつけることである。
真空中で引力を生むミクロな起源として仮想的な$\Sigma$粒子への遷移を伴う「$2\pi$交換プロセス」が有力視される。引力全体への定量的寄与を確定させるため、反応頂点における$\Lambda N$-$\Sigma N$結合力をE90実験の「$\Sigma N$カスプ分光」によって正確に抽出する。
一方、高密度媒質中ではパウリ排他律によってこの遷移が阻害（パウリブロッキング）され引力が消滅し、クォークの重なりによる短距離斥力芯が支配的になるという仮説がある。本研究では、中心密度が標準密度の2倍（$2\rho_0$）に達する$\alpha$粒子に着目し、「$\alpha\Lambda$フェムトスコピー」によって極小ソースから斥力芯を直接抽出する。三輪らの散乱実験による真空中基準力と理論計算を統合し比較することで、斥力化仮説の真偽を明らかにする。

\subsubsection{研究内容の詳細と到達目標（どこまで明らかにするか）}

\noindent
\textbf{1. $\Sigma N$カスプ分光（E90実験）による結合力の決定（分担者：市川、谷田）}\\
半世紀前の反応測定において、$\Sigma N$生成閾値（$\sim 2130\,\text{MeV}$）直下に極めて鋭い「$\Sigma N$カスプ」構造が観測された。これは生成された$\Sigma N$対が終状態相互作用によって中間状態で $\Sigma N \to \Lambda N$ へ強くチャンネル転換することに起因して生成される。
遷移振幅 $T$ は散乱長 $A_\Sigma = a + i b$ を用いて $T \propto \operatorname{Im}(A_\Sigma)/(1 - i k_\Sigma A_\Sigma)$ と記述される。虚部 $b$ は本研究が標的とする**「$\Lambda N$-$\Sigma N$間の結合力」**を直接表す物理量である。
反応率 $R \propto |T|^2$ は閾値上・下において以下のように展開される。
\begin{equation}
    R_{E > M_\Sigma + M_N} \sim \frac{4\pi b}{(1 + k_\Sigma b)^2 + (k_\Sigma a)^2}
\end{equation}
\begin{equation}
    R_{E < M_\Sigma + M_N} \sim \frac{4\pi b}{(1 - |k_\Sigma| a)^2 + (|k_\Sigma| b)^2}
\end{equation}
閾値前後で現れる特徴的かつ非対称なスペクトル形状は散乱長に対して極めて高い感度を有するため、スペクトル形状を高精度測定することで実部と虚部を同時に分離して導出できる。
J-PARC E90実験では、K1.8ビームラインとS-2Sスペクトロメータにより運動量を極限精度で測定し、カスプ質量スペクトルを分解能 $\sigma \sim 0.4\,\text{MeV}$ で捉える。飛跡検出器HypTPCで崩壊対を同時検出し背景事象を排除することで、散乱長を $0.3\,\text{fm}$ の精度で確定させる。

\vspace*{1zw}
\noindent
\textbf{2. $\alpha\Lambda$フェムトスコピーによる高密度媒質中短距離力の解明（分担者：市川・佐甲・大浦）}\\
${}^5_\Lambda\mathrm{He}$ の束縛エネルギー積分値のみでは短距離ポテンシャルの詳細形状に甚大な不定性が残る。斥力化シナリオの実証には強い短距離斥力芯の実在証明が不可欠である。
ハドロン対の運動量相関から相互作用を調べるフェムトスコピーにおいて、相関関数 $C(k^*)$ の精密な理論計算（神野・神谷）によれば、大ソースサイズでは長距離力が支配し全モデルが縮退するが、極小ソースサイズ $R = 1\,\text{fm}$ の環境下では波動関数が短距離ポテンシャルの違いを鋭く拾い上げ、斥力芯の有無で決定的な差異が生じることが判明した。
本研究では、複合核生成断面積が爆発的に増大する中高エネルギー領域に着目し、J-PARC E88実験（$pA$衝突）およびRHIC-STAR実験（$AA$衝突）で測定を実行する。特にE88実験では全事象を記録可能な「ストリーミングDAQシステム」を導入し、極小サイズが保証される反応からの貴重なデータを確実に捉える。

\vspace*{1zw}
\noindent
\textbf{3. 実験データを基にした理論研究（分担者：神野・神谷）}\\
得られた「結合力」および「斥力」データをカイラル有効場の理論（Chiral EFT）へ統合し、高密度領域まで適用可能な包括的相互作用モデルを構築する。中性子星の最大質量予測を行い、パズル解決の可否に最終的な理論的結論を下す。

\subsection{本研究の学術的独自性と創造性}
ハイペロンを標的とすることが不可能な散乱実験の限界に対し、重水素標的を用いて**「特定のスピン状態を選択した散乱長の決定」**を実現する点が第1の独自性である。
第2の創造性は、フェムトスコピーを**「原子核（$\alpha$粒子）とハドロンの相関」**に拡張する点にある。中心密度 $2\rho_0$ を持つ実体的な原子核プローブと極小ソースの相乗効果で長距離力を抑え込み、超高密度下の短距離斥力芯シグナルのみを劇的に「増幅」させて抽出する。

\subsection{関連する国内外の研究動向と本研究の位置づけ}
真空中における長距離力の検証はALICEやSTAR実験が進め、短距離力の直接測定は三輪らの散乱実験（HYPS/E86）が進行中である。しかしパズル解決には「媒質中での相互作用変化」の解明が不可欠である。本研究は静止標的の特性で極小ソースを実現し、引力源泉と高密度斥力芯を同時に決定する。世界的な進展と相補的な関係を築きつつ、斥力化シナリオによる決着の決定打を放つ**唯一無二の国際的ポジション**にある。

\subsection{本研究の目的を達成するための準備状況}
主要拠点であるJ-PARC K1.8ビームラインは大強度ビームの利用が保証されている。
\begin{itemize}
    \item \textbf{E90実験}：Stage-1承認およびTDR提出を完了し、S-2S検出器の精密校正や真空化等のハード改良を進め、2028年のデータ取得へ準備は最終段階にある。
    \item \textbf{連携体制}：分担者の神谷はALICE実験の相関解析で実績があり連携を維持する。HYPS実験用TPC開発も順調である。
    \item \textbf{E88/STAR実験}：E88はストリーミングDAQ導入用の回路開発に着手し、2026年に追加提案書を提出予定。STAR-BESデータは全取得を完了し、専任ポスドク雇用により連携して即座に解析を開始する体制である。
\end{itemize}
