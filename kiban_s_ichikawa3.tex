	本研究の目的は、極限状態の超高密度天体である中性子星の内部構造を決定づける\textbf{最大の難問「ハイペロンパズル」}の解決に向け、加速器実験を用いた独創的な新手法により、ハイペロン（$\Lambda$粒子）と核子の間に働く\textbf{短距離相互作用を解明すること}である。

\subsection{本研究の学術的背景や着想に至った経緯、研究課題の核心をなす学術的「問い」}
\subsubsection{学術的背景：中性子星とハイペロンパズルの陥穽}
宇宙に浮かぶ巨大な原子核とも呼ぶべき天体「中性子星」の中心部には、標準原子核密度（$\rho_0 \sim 0.17\,\text{fm}^{-3}$）の数倍に達する極限的な超高密度環境が実現している。天体内部に向かって密度が高まり核子のフェルミエネルギーが増大すると、ストレンジクォークを含むハイペロン（特に$\Lambda$粒子）を生成した方が系の全エネルギーが低くなるため、標準密度の2〜3倍程度の領域から星の中心部におけるハイペロン出現は\textbf{熱力学的な必然}となる。これまでの$\Lambda$ハイパー核精密分光研究から、標準密度において$\Lambda$粒子が感じるポテンシャルの深さは約 $-30\,\text{MeV}$ であることがわかっており、$\Lambda$粒子と核子間には明確な引力が働くことが実証されている。したがって、$\Lambda$粒子は中性子星の中心部に極めて出現しやすい状態にある。

しかし、標準的な核理論モデルに基づく状態方程式（EoS）の予測によれば、ハイペロンの出現はフェルミ圧を著しく減少させ、\textbf{状態方程式を劇的に軟化させる}。その結果、近年の天体観測によって確固たる証拠が得られている\textbf{太陽質量の2倍（$2M_\odot$）もの重い中性子星}の重力を支えることができず、ブラックホールへと崩壊してしまうという深刻な理論的矛盾が生じる。これが現代核物理学の最大難問「ハイペロンパズル」である。ミクロなハドロン間相互作用の性質に基づき、この天体スケールの矛盾を根本的に解決しなければならない。

\subsubsection{パズル解決に向けた2つの有力な仮説と着想に至った経緯}
現在、このハイペロンパズルを解決し得るシナリオとして、以下の2つの有力な仮説が議論されている。
\begin{enumerate}
    \item \textbf{相互作用の斥力化シナリオ}：標準密度においては引力的である$\Lambda N$相互作用が、中性子星深部のような高密度環境下においては相互作用の短距離成分に由来する強い「斥力」へと転化するため、ハイペロン出現自体が抑制されるか硬いEoSが維持されるという仮説。これを立証するためには、高密度媒質中で相互作用が斥力化するメカニズムを定量的かつミクロに解明する必要がある。
    \item \textbf{クォーク物質への相転移シナリオ}：ハイペロンが出現する直前の極限密度においてハドロンからの非閉じ込め相転移が起き、ストレンジクォークを含む解放されたクォーク物質が形成されることで十分な縮退圧を維持するという仮説。
\end{enumerate}

本研究の核心的な問いは\textbf{「核物理は中性子星におけるハイペロンパズルを解くことができるのか？」}にある。極限環境でのクォーク相転移シナリオは理論的に極めて魅力的であるが、中性子星深部の「低温・超高密度環境」を地上実験室の重イオン衝突等で直接再現し証明することは現在の技術では極めて難しい。
そこで申請者は\textbf{「高密度環境で$\Lambda$粒子が強い斥力を感じ得るのか？」}という実験的に検証可能な問いに着想を得た。加速器実験の多角的な精密データに基づき斥力化シナリオの正否を徹底検証し、もし斥力化メカニズムのみでは重い中性子星を支えられないと判明した場合は、状況証拠的（背理法的）にクォーク相転移シナリオの正当性を極めて高く裏付けることができる。\textbf{短距離斥力芯の極限探求こそが、相転移立証に向けた最も確実かつ鋭利なアプローチ}である。

\subsection{本研究の目的、研究内容、独自性と創造性、どこまで明らかにするか}
\subsubsection{本研究の目的：相互作用の短距離成分の解明}
本研究の目的は、真空中および高密度媒質中における$\Lambda N$相互作用の\textbf{短距離成分を実験的に解明し、ハイペロンパズルに決着をつけること}である。
真空中で強い引力を生むミクロな起源として、$\Lambda$粒子が中間状態で同種クォーク構造を持つ$\Sigma$粒子へと遷移しながら2つの$\pi$中間子を交換する「$2\pi$交換プロセス」が有力視されている。このメカニズムの引力全体への定量的寄与を確定させるため、反応頂点における$\Lambda N$-$\Sigma N$結合力をE90実験の「$\Sigma N$カスプ分光」によって正確に抽出する。

\subsubsection{研究内容の詳細と到達目標（どこまで明らかにするか）}

\noindent
\textbf{1. $\Sigma N$カスプ分光（E90実験）による結合力の決定（分担者：市川、谷田）}\\
半世紀前の反応測定において、$\Sigma N$生成閾値（$\sim 2130\,\text{MeV}$）直下に極めて鋭い「$\Sigma N$カスプ」構造が観測された。図\ref{fig:cusp}左に示すように、これは反応で生成された$\Sigma N$対が終状態相互作用によって中間状態で強く $\Sigma N \to \Lambda N$ へとチャンネル転換を起こすことに起因して生成される。
遷移振幅 $T$ は$\Sigma N$間の複素散乱長 $A_\Sigma = a + i b$  と相対運動量 $k_\Sigma$ を用いて以下のように記述される。
\begin{equation}
    T(\Sigma N \to \Lambda N) \propto \frac{\operatorname{Im}(A_\Sigma)}{1 - i k_\Sigma A_\Sigma}
\end{equation}
ここで$\Sigma N$間の散乱長の虚部 $b$ は、本研究が標的とする\textbf{「$\Lambda N$-$\Sigma N$間の結合力」}の強さを直接表す極めて重要な物理量である。
反応率 $R \propto |T|^2$ は質量欠損 $M$ が閾値上・下においてそれぞれ以下のように展開される。
\begin{equation}
    R_{M > M_\Sigma + M_N} \sim \frac{4\pi b}{(1 + k_\Sigma b)^2 + (k_\Sigma a)^2}
\end{equation}
\begin{equation}
    R_{M < M_\Sigma + M_N} \sim \frac{4\pi b}{(1 - |k_\Sigma| a)^2 + (|k_\Sigma| b)^2}
\end{equation}
閾値前後で現れる特徴的かつ非対称な質量スペクトル形状は、散乱長の実部および虚部に対して極めて高い感度を有するため、スペクトル形状を高精度測定することで両者を同時に分離導出することが可能となる。図\ref{fig:cusp}右に示すように、散乱長パラメータの違いはスペクトルのピーク位置や裾野の広がりとして明瞭な差異を生む。
本実験を立ち上げた実験責任者である市川および谷田の主導のもと、J-PARC E90実験では、大強度K1.8ビームラインと高運動量分解能を誇るS-2Sスペクトロメータにより運動量を高精度で測定し、カスプ質量スペクトルを\textbf{世界最高のエネルギー分解能 $\sigma \sim 0.4\,\text{MeV}$} で捉える。さらに標的周囲に配置した飛跡検出器HypTPCで崩壊対を同時検出し、背景事象を排除することで、\textbf{散乱長を $0.3\,\text{fm}$ の精度で確定}する。

\noindent
\textbf{2. $\alpha\Lambda$フェムトスコピーによる高密度媒質中短距離力の解明（分担者：市川・佐甲・大浦）}\\
${}^5_\Lambda\mathrm{He}$ 等のハイパー核束縛エネルギーの積分値のみからは短距離ポテンシャルの詳細な芯形状に甚大な不定性が残り、高密度斥力化シナリオの実証には\textbf{短距離斥力芯の実在証明が不可欠}である。
ハドロン対の相対運動量相関から相互作用を調べるフェムトスコピーにおいて、相関関数 $C(k^*)$ の精密な理論計算（神野・神谷）によれば、重イオン衝突等で実現される大ソースサイズ環境下では長距離力が支配して全相互作用モデルの予測が縮退してしまう。しかし、極小ソースサイズ $R = 1\,\text{fm}$ の極限環境下においては、ハドロン対の波動関数が短距離ポテンシャルの違いを鋭く拾い上げ、斥力芯の有無によって相関関数に決定的な差異が生じることが判明した。

本研究で推進するフェムトスコピーと散乱実験の関係を図\ref{fig:femto}に示す。中心密度が標準密度の2倍（$2\rho_0$）に達する実体的な高密度プローブである $\alpha$ 粒子を活用し、極小ソースサイズが保証される反応環境と組み合わせることで長距離力の寄与を効果的に抑え込み、超高密度下における\textbf{短距離斥力芯のシグナルのみを劇的に顕在化させて抽出する}。
本研究では、複合核生成断面積が爆発的に増大する中高エネルギー領域に着目し、E88実験の実験責任者である佐甲を中心として、J-PARC E88実験（$pA$衝突）およびRHIC-STAR実験（$AA$衝突）における相補的な測定を実行する。特にE88実験のデータ収集においては、同実験のDAQシステム開発を担当する特別研究員（PD）の大浦が主導し、トリガーバイアスを排除して全事象を柔軟に記録可能な次世代技術\textbf{「ストリーミングDAQシステム」}を独自導入することで、貴重な相関データを確実に捉える。真空中における$\Lambda p$散乱実験から得られる裸のポテンシャルを強固な比較基準とし、媒質中での相互作用変化を決定づける。

\subsection{本研究の学術的独自性と創造性}
ハイペロンをビームや標的とすることが不可能な直接散乱実験の限界に対し、重水素標的を用いて終状態相互作用を測定し\textbf{「特定のスピン状態を選択した散乱長の決定」}を実現する点が第1の独自性である。
第2の創造性は、フェムトスコピーの手法を従来のハドロン対から\textbf{「原子核（$\alpha$粒子）とハドロンの相関」}へと飛躍的に拡張する点にある。実体的な高密度プローブである $\alpha$ 粒子と極小ソースサイズの相乗効果により、長距離ポテンシャルの寄与を抑え込み、超高密度下における\textbf{短距離斥力芯のシグナルのみを劇的に「増幅」させて抽出する}ことに成功する。

\subsection{関連する国内外の研究動向と本研究の位置づけ}
真空中における長距離相互作用の検証はCERN-ALICEやBNL-STAR実験が強力に進め、真空中の短距離力の直接測定は三輪らの散乱実験プロジェクト（HYPS/E86）が進行中である。しかしパズルの解決には「媒質中での相互作用の密度変化」の解明が不可欠である。本研究は静止標的実験の特性を活かして極小ソースを実現し、真空中の引力源と高密度下の斥力芯を同時に決定する。これら国際的な研究展開と相補関係を築きつつ、\textbf{斥力化シナリオによる決着の決定打を放つ唯一無二の国際的ポジション}を確立している。

\end{itemize}
